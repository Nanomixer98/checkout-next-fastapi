## Run backend locally
Requirements:
- Python 3.9+ 

### Install dependencies
```bash
pip install -r requirements.txt
# or if you have trouble with the previous command
pip install --trusted-host pypi.org --trusted-host files.pythonhosted.org -r requirements.txt
```
